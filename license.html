Licensing FAQ

What does "white-label" mean?

White-label means you can replace our logo, colors, and branding with your own. The game still runs on our engine, but it looks like it belongs to your platform. Players see your brand, not ours.

Do I need a server to run the game?

No. Mahjong Quiz is a pure client-side game. All game state is stored in the player's browser (localStorage). You can host it on any static file server (Netlify, Vercel, S3, GitHub Pages) or embed it via an iframe. Zero server costs.

Can I customize the yaku, rules, or difficulty?

Pro and Enterprise tiers include configuration files that let you adjust which yaku are available, the round scaling, score targets, and relic pools. Enterprise tier includes source code access for deeper customization.

Does it work on mobile?

Yes. The game is fully responsive and works on mobile browsers. It's also a PWA (Progressive Web App), so players can install it to their home screen for a native-app feel.

What analytics does the game include?

The game ships with Umami analytics integration (privacy-friendly, cookie-free, GDPR-compliant). You can swap in your own analytics provider (Google Analytics, Plausible, etc.) or disable analytics entirely.

How do updates work?

Each tier includes a period of free updates. Starter: 1 year. Pro: 2 years. Enterprise: lifetime. Updates include new yaku, relics, balance changes, and bug fixes. After the update period, you can purchase an extension.

Can I add my own ads to the game?

Yes. All tiers allow you to add your own ad placements (AdSense, banner ads, rewarded video) to the game. We provide ad slot placeholders in the game layout. You keep 100% of your ad revenue.
