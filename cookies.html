Cookie & Local Storage Policy

Last updated: July 18, 2026

The short version: The Game uses localStorage to save progress and preferences, plus Cache Storage for offline assets. We do not currently load an analytics or advertising provider. The first-visit notice records only that you dismissed it.

1. Our Storage Approach

Mahjong Quiz is designed to work without accounts. The Game saves progress in browser localStorage, not in an account database. We do not set our own login, account, or session cookies.

The current site does not load an analytics or advertising provider. If we introduce non-essential storage or tracking later, we will update this policy and add consent or opt-out controls where required.

2. What Are Cookies?

Cookies are small text files stored on your device by websites you visit. They are widely used to track users across sessions and sites, enable advertising networks, and store authentication state. Common types include:

  • Tracking cookies - Follow activity across websites.
  • Advertising cookies - Enable ad serving, measurement, fraud prevention, frequency capping, and sometimes personalization.
  • Session cookies — Maintain login state (we don't use these; no accounts).
  • Preference cookies — Remember settings (we use localStorage instead).

4. LocalStorage Details

The Game uses the browser's localStorage API to save your progress. localStorage is fundamentally different from cookies:

  • Not automatically attached to requests: Browsers do not send localStorage values with every HTTP request as they do with cookies.
  • Persistent: These values normally remain until the site or user clears them.
  • Origin-scoped: Other website origins cannot directly read this site's localStorage.
  • Current use: The current application code reads these values for gameplay and does not send them to us.

The following localStorage keys are used:

  • mjrg_run — Current run state (round, score, relics, custom tiles)
  • mjrg_meta — Meta-progression (total runs, wins, best score, achievements)
  • mjrg_settings, mjrg_sound_muted — Gameplay and sound preferences
  • mjrg_yaku_bonuses — Active yaku boost bonuses
  • mjrg_leaderboard, mjrg_daily, mjrg_mistakes — Local scores, daily-challenge state, and review history
  • mjrg_onboarded, mjrg_beginner_done, mjrg_tutorial_seen, mjrg_normal_done — Tutorial and mode-completion state
  • mjrg_cookies_accepted — Acknowledgement that the first-visit storage notice was dismissed

As a Progressive Web App (PWA), the Game registers a service worker that caches static assets (HTML, CSS, JavaScript, images) in a Cache API storage. This enables offline play after the first visit. This cache is also browser-local, not transmitted, and not a cookie. You can clear it via your browser's "Site data" settings.

You can clear these values at any time via your browser's site-data settings or developer tools (Application → Local Storage / Cache Storage).

5. Analytics and Advertising

No analytics or advertising provider is currently loaded on the public site. The codebase includes an inactive analytics integration point and empty advertising layout placeholders, but they do not contact a provider. If either feature is enabled later, this policy will be updated and any legally required choices will be provided.

6. External Requests

The current site makes ordinary web requests to third-party infrastructure:

  • Google Fonts — Content pages request font files from Google's servers, which receive the request's ordinary technical information. See Google's Privacy Policy.
  • Vercel — Hosts and serves the site and may process request or security information. See Vercel's Privacy Policy.

These external requests are separate from the locally stored game-progress values described above.

7. Server Logs (Hosting Provider)

Vercel may process standard request and security information when it serves the Game. Depending on its systems and settings, that information may include:

  • IP address (used for security/abuse detection, not linked to identity)
  • Timestamp of the request
  • URL requested and HTTP method
  • User-Agent string (browser and operating system)
  • Referrer URL
  • Response status code

Server logs are not browser cookies. Their handling and retention depend on Vercel's services, settings, and policies; we do not promise a retention period that we do not control. See Vercel's Privacy Policy.

8. How to Manage Your Data

You can manage localStorage, Cache Storage, and cookies through your browser:

8.1 Browser-Specific Instructions

  • Google Chrome — Settings → Privacy and security → Clear browsing data → select "Cookies and other site data" and "Cached images and files."
  • Mozilla Firefox — Settings → Privacy & Security → Cookies and Site Data → Clear Data.
  • Apple Safari — Preferences → Privacy → Manage Website Data → Remove All.
  • Microsoft Edge — Settings → Cookies and site permissions → Manage and delete cookies and site data.
  • Brave — Settings → Additional Settings → Privacy and security → Clear browsing data.

8.2 Quick Actions

  • To clear game progress and preferences: Clear this site's data through your browser as described above.
  • To block remote fonts: Use a browser privacy setting or content blocker. The site will fall back to system fonts.
  • To disable service worker / PWA: Use your browser's developer tools (Application → Service Workers → Unregister). The Game will still work but will not be available offline.
  • To browse privately: Use Incognito/Private Browsing mode. Your game progress will not be saved when you close the window.

9. Your Choices and Applicable Rules

Depending on where you live, laws such as the EU ePrivacy rules or UK PECR may require consent before non-essential storage or tracking is used. The current notice's "Got it" button only records that the notice was dismissed; it is not presented as consent to analytics or advertising, because neither is currently loaded.

  • Blocking or clearing local storage may reset progress, settings, and offline files.
  • We do not currently sell or share personal information for cross-context behavioral advertising.
  • See our Privacy Policy for information about data handling and regional rights.
